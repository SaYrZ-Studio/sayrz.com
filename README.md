# sayrz.com

## Project Structure

```
config.json
index.js
package.json
README.md
webroot/
		index.html
		docs.html
```

## Usage

- Place HTML files in the `webroot` directory
- Run Node.js code from `index.js`
- Add dependencies to `package.json`

## Commands

- Start server:
	```bash
	npm start
	```

- (Optional) "If" you "add" a `dev` script to `package.json`, run:
	```bash
	npm run dev
	```
